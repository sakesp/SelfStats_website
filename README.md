# SelfStats Website

A simple static website that can be published with GitHub Pages.

## Files

- `index.html` is the page GitHub Pages will serve first.
- `styles.css` controls the layout and visual design.
- `script.js` adds the current year in the footer.
- `assets/selfstats-dashboard.png` is the hero background image.

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Push this folder to that repository.
3. In GitHub, open the repository settings.
4. Go to Pages.
5. Set the source to deploy from a branch.
6. Choose the `main` branch and the `/ (root)` folder.
7. Save. GitHub will show the public Pages URL after it finishes deploying.

If your repository is named `USERNAME.github.io`, the site will usually be at
`https://USERNAME.github.io/`. Otherwise, it will usually be at
`https://USERNAME.github.io/REPOSITORY-NAME/`.
